# atividade-algoritmos-ordenacao
Este projeto em C analisa o desempenho dos algoritmos Bubble Sort e Selection Sort em vetores de 100, 1000 e 10000 elementos em ordem decrescente. O código utiliza a função clock() para medir o tempo e contabiliza rigorosamente todas as comparações e movimentações, comparando a eficiência de ambos no cenário de pior caso.
